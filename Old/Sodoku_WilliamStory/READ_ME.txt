When the program runs, it will open an open file dialog window
that will prompt for the location of the txt file
that contains the Sudoku puzzle. 

The puzzle with follow the following format:

,7,,5,,,,9,
4,,3,,,,,7,
5,6,,,,1,,,3
,3,,,,,7,,
6,,,,5,,,4,8
9,,5,7,,,,,6
,2,1,6,,,,8,
,9,,,,,5,3,
,,,,3,,,,

After it loads, the button labeled "solve!"
will solve the puzzle once clicked. After
the program has either solved the puzzle
or given up in attempting to solve,
a message box displaying the number of solved
boxes will display.
